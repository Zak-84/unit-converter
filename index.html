Unit Converter Bookmarklet

Convert between millimeters and inches instantly on any webpage

Install the Bookmarklet

Drag this button to your bookmarks bar:

Unit Converter

(Drag the button above to your bookmarks bar)

How to Install

  1. Make sure your browser's bookmarks bar is visible.
    • In Chrome: View → Always Show Bookmarks Bar (or press Ctrl+Shift+B)
    • In Firefox: View → Toolbars → Bookmarks Toolbar
    • In Safari: View → Show Favorites Bar
    • In Edge: Settings → Appearance → Show favorites bar
  2. Drag the "Unit Converter" button above to your bookmarks bar.
  3. Visit any webpage with measurements in mm or inches.
  4. Click the "Unit Converter" bookmark to automatically convert all measurements on the page.

What It Does

This bookmarklet scans the entire webpage for measurements in:

  • Millimeters (e.g., "42mm" or "42 mm")
  • Inches (e.g., "1.65 inches" or '1.65"')

It then adds the converted value right next to the original measurement in bold red text.

Example:

Original text: Scale length: 648 mm

After clicking bookmarklet: Scale length: 648 mm 25.51 inches

Original text: Nut width: 1.6875"

After clicking bookmarklet: Nut width: 1.6875" 42.86 mm

Troubleshooting

  • Bookmarklet not working? Some websites have security settings that prevent bookmarklets from running. Try a different website.
  • Missing some measurements? The bookmarklet looks for common patterns like "42mm" or "1.65 inches". If a measurement is formatted uniquely, it might be missed.
  • Page loads slowly after using the bookmarklet? The script scans all elements on the page, which might cause performance issues on very large pages.
